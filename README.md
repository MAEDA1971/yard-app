# プレキャスト製品ストックヤード DXF表示

## 動作確認方法

### 1. VS Codeでローカルサーバーを起動する

1. VS Codeのメニューから「ターミナル」→「タスクの実行」をクリックします。
2. リストから `Start Local Server` を選びます。
3. ターミナルに `Serving HTTP on 0.0.0.0 port 8000` のような表示が出たら起動完了です。

### 2. ブラウザで開く

1. ブラウザで次のURLを開きます。

   `http://localhost:8000/index.html`

2. これでDXFファイルが読み込まれるはずです。

## 重要な注意

- `index.html` を直接開く（`file://...`）と、ブラウザはDXFファイルを読み込めません。
- このため、必ず上の手順でローカルサーバーを起動してください。

## DXFファイルの場所

- フォルダ: `public/dxf`
- ファイル名: `yard-map.dxf`

このファイルを差し替えると、ブラウザを再読み込みしたときに表示が更新されます。
